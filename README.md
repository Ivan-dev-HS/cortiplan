# CortiPlan

Aplicación de Hoja de Instalación y Corte para Canotex Textil.

## Instalar en GitHub Pages

1. Crea un repositorio en GitHub (ej: `cortiplan`)
2. Sube todos estos archivos
3. Ve a Settings → Pages → Source: main branch
4. Tu app estará en: `https://TU_USUARIO.github.io/cortiplan`

## Instalar en escritorio / móvil

1. Abre la URL en Chrome o Edge
2. Haz clic en "Instalar en escritorio" (sidebar) o en el icono de instalar del navegador
3. CortiPlan se instalará como app nativa con su icono
